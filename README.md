# Food-delivery

## Features

1. User Registeration and Authentication
2. Restaurant management
3. Menu management
4. Order management
5. Cart management
6. Payment management
7. Customer management
8. Dashboard [Restaurant, System]

<br/>

### 1. User Registeration and Authentication

- Login
- Sign up
- Forget password
- Email verification or OTP verification
- User/Customer profile
- Log out
- Social media authentication
- Enable/Disable account

### 2. Restaurant/Menu Management

- Add restaurant
- Update restaurant
- Enable/Disable restaurant
- Delete restaurant
- View all restaurants
- Create new menu
- Update menu
- Delete menu
- View menu history list
- View list of menus
- Search restaurant
- Search menu item
- Top rating restaurants

### 3. Order Management

- Place an order
- Order cancel by customer or restaurant
- Order status update
- Notify customer when order status update
- Order confirmation by email/SMS

### 4. Cart Management

- Add to cart
- Remove item from cart
- Update quantities
- Clear all cart
- Checkout
- View cart

### 5. Payment Management

- Payment integration with 3rd party
- Multiple payment methods
- View payment transactions
- Payment status
- Payment error handling
- Audit payment integration
- Generate payment receipt
- Payment verification and validation

### 6. Customer management

- List orders history
- Current order status
- Address management [Multiple delivery address]
- Prefered payment settings [Visa, Cash on delivery]
- Account activate/deactivate
- Ratings and comments

### 7. Dashboard [Restaurant - System]

#### **System**

- Count of restaurants
- Count of customers
- Count of active customers
- Daily count orders
- Total count orders
- Daily canceled orders
- Total canceled orders
- Daily amount of transactions
- Total amount of transactions
- Generate daily transaction report
- Generate monthly transaction report

#### **Restaurant**

- Daily count orders per restaurant
- Daily count orders not delivered by restaurant
- Total count order per restaurant
- Daily cancelled orders per restaurant
- Total cancelled orders per restaurant
- Daily amount of transactions per restaurant
- Total amount of transactions per restaurant
- Generate daily transaction report per restaurant
- Generate monthly transaction report per restaurant
